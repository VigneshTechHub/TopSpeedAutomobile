# TopSpeed Auto Mobile

**TopSpeed Auto Mobile** is a full-stack ERP system designed for managing vehicle service operations. Built entirely with **ASP.NET Core MVC**, it features a responsive frontend using **Razor views**, **Bootstrap**, **Tailwind CSS**, **HTML**, **CSS**, and **JavaScript**. The backend is powered by **ADO.NET** and **Entity Framework Core**, following a clean **Repository Pattern** and **Model-View-Controller (MVC)** architecture.

## 🔧 Features
- Vehicle showcase module with price, features, and service details (no purchase flow)
- Service booking, billing, and customer management workflows
- Role-based authentication using **JWT tokens** for Admin and Staff
- Responsive UI with Razor views and reusable Bootstrap/Tailwind components
- Backend logic implemented using **ADO.NET stored procedures** and **EF Core**
- Clean separation of concerns using Repository & Service layers

## 🧠 Tech Stack
**Frontend**: Razor • Bootstrap • Tailwind CSS • HTML • CSS • JavaScript  
**Backend**: ASP.NET Core MVC • ADO.NET • Entity Framework Core • SQL Server  
**Security**: JWT Authentication • Role-based access  
**Architecture**: MVC • Repository Pattern • Stored Procedures

## 📦 Notes
This project focuses on service management and vehicle showcasing. It does not include a buying or checkout module. All modules were designed and delivered with real-world business logic, secure workflows, and staging-level validation.
